    Webdesign

    I Flow 3 skulle vi bygge vores egen hjemmeside, vi startede med nogle enkle opgaver, hvor vi skulle kopier et eksempel, således at vi forstod kodesproget før vi skulle lave vores egen hjemmeside.

    I denne flow brugt vi HTML og CSS til at programmer vores hjemmesider ved brug af Visual Studio Code. opgaverne at lave et Visuelt Koncept til digitale og printede medier.

    Hvad er HTML?
    HTML står for 'Hyper Text Markup Language', html styrer det grundlæggende struktur og konstruktion bag en website. En hjemmeside kan ikke laves eller fungere, uden html.

    Hvad er CSS
    CSS står for 'Casading Style Sheets', en CSS-fil kan ikke stå alene og den forbindes med html filen. CSS formatér og omklæder HTML siden til at se professionelt og flot ud.

    Hvad er sammenhængen mellem HTML og CSS?
    Det kan godt være, at det grundlæggende struktur og konstruktion er styret af HTML, men CSS klæder html siden med f.eks. farver og format. Alle farvernes default på html er sort, man kan ikke formatér, ændre farve, flyt, lave animationer og alle de andre lækre ting man finder på en hjemmeside den CSS. Derfor går de hånd-i-hånd, og er derfor vigtigt at man kan forstår og programmer både i HTML og CSS.

    En af de vigtigste krav til opgaven, det var Mobile-First. Mobile-first indexing går betyder, at din hjemmeside er mobil-venlig, således at folk bruger den på deres mobil, så justerer formattet af hjemmesiden således at det bliver et fornufrigt hvis iks bedre oplevelse for mobilbrugerne. Mange bruger nemlig deres mobiler nå de browser, og det er altid optimalt at have en hjemmeside som kan bruges både på computer og mobil.
